# Lobby Locator

![GitHub License](https://img.shields.io/github/license/james-minor/lobby-locator)
![GitHub last commit (branch)](https://img.shields.io/github/last-commit/james-minor/lobby-locator/master)

A Discord bot to help you find people to play Steam games with.

## How it Works

The Lobby Locator bot allows users to connect their Steam IDs to the bot database. Users who have registered their 
Steam ID can then ping other connected users with the bot, letting users who own the game get notified about potential
lobbies.

## Features
- Pinging users based on game-ownership
- Blacklisting of games a user does not wish to be notified for

## Running the Bot

Currently, the only option to run Lobby Locator is to self-host your own version of the application.
